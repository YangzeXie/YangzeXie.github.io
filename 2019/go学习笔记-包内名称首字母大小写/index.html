Go学习笔记 包内名称首字母大小写

Written by YangzeXie with ♥ on 17 November 2019 in 1 min

Go语言的包内函数首字母有大小写的区别，如果一个函数名字以大写字母开头，那么它就是已导出的，如果一个函数名字以小写字母开头，它就是未导出的，所谓已导出与未导出的区别在于该函数是否可以在这个包外进行调用，下面通过代码进行对比。

package main

import (
	"fmt"
	"math"
)

func main() {
	fmt.Println(math.Pi) //math.Pi为已导出的，可在包外进行调用，math.pi为未导出的，只能在包内部进行调用
}
